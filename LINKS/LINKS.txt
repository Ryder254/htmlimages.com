<h2>HTML links (hyperlink)</h2>
<p>Html links are hyperlinks.<br>You can click a link to jump to another page</p>
<h3>HTML links syntax</h3>
<p>The HTML <mark><b>&lt;a&gt;</b></mark> tag defines a hyperlink.It also has the following syntax</p>
<p><b><mark><ins>&lt;a href="url">link text&lt;/a&gt;</ins></mark></b></p>
<p>Only the link text is visible in the web page<br>By default, links will appear as follows in the web page</p>
<ul style="list-style-type:disc">
 <li>An unvisited link will be underlined and blue</li>
 <li>A visited link will appear as underlined and purple in  color</li>
 <li>An active link will be underlined and red in color</li>
</ul>
<p><span style="color:purple;background-color:rgba(140,140,140,0.3);font-family:CASTELLAR;"><b><mark>Tip!</mark>Links can of course be styled with CSS, to get another look!</b></span></p>
<h3>HTML links- The target attribute</h3>
<p> By default the linked page will be displayed in the current browsing window.<br>To change this you must specify another target for the link.The <mark><b>target</b></mark> attribute can have the following targets</p>
<dl type="i">
 <dt><b><mark>___self</mark></b>-default.</dt>
 <dd>Opens the document in the same window/tab as it was clicked<dd>
 <dt><b><mark>___blank</mark></b>.</dt>
 <dd>Opens the clicked document in a new window/tab</dd>
 <dt><b><mark>___parent</mark></b></dt>
 <dd>Opens the clicked document in a the parents frame.</dd>
 <dt><b><mark>___top</mark></b></dt>
 <dd>Opens the clicked document in the full body of the window</dd>
</dl>
