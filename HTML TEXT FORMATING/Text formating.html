HTML TEXT FORMATING

HTML TEXT FORMATING elements are the elements that defines how the text appears to the users

These elements may include;

b
This element is used to put some words in a sentence in bold
strong
The element is used to show and important text.
The word where the element is used appears to be bold but when a screen reader is used, the word is read as an important text
i
The element is used to italitise words in a paragraph or a heading
em
The element is used to emphasize on a text.
The word where the element is used appears to be italised but when a screen reader is used the word is emphasized
mark
This emelent is used to highlight a word in a paragraph
small
The element is used to make texts in a paragraph smaller that the others
del
THe element is used to put a line across a word or texts to mean that the works are deleated
ins
The element shows that the word or a set of words have beeen inserted in a paragraph or a heading
sub
The element defines a subscript
sup
The element defines a superscript
